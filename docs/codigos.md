# Códigos
- Use o conceito de **Código Limpo (Clean Code)** na escrita do código.
- Use variáveis e funções com nomes expressivos.
- Não abrevie as condições
- Não use `var` use sempre o tipo explícito.
- Todas as variáveis, funções e comentários devem ter nomes em português.
- Não use comentários desnecessários.
- Use formatação `.ToString("C")` para máscara de dinheiro.
